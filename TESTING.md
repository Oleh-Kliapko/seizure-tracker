# Чеклист тестування — SeizureTracker

Проходь цей список вручну перед кожним новим релізом або після великих змін.
Тестуй на реальному iPhone, а не тільки на симуляторі.

---

## Авторизація

- [ ] Реєстрація через email/пароль — новий акаунт створюється
- [ ] Реєстрація з вже існуючим email — показується помилка
- [ ] Реєстрація зі слабким паролем — показується помилка
- [ ] Реєстрація — приходить лист верифікації на пошту
- [ ] Логін через email/пароль — успішний вхід
- [ ] Логін з невірним паролем — показується помилка
- [ ] Логін через Google — успішний вхід
- [ ] Логін без інтернету — показується адекватна помилка
- [ ] Закрити додаток і відкрити знову — залишаєшся залогінений
- [ ] Логаут — перекидає на екран логіну
- [ ] Після логауту повторний вхід — працює

---

## Профіль

- [ ] Відкрити вкладку "Особисті дані" — завантажуються збережені дані
- [ ] Змінити ім'я / телефон / адресу — зберігається після закриття екрану
- [ ] Завантажити фото аватара — відображається
- [ ] Замінити фото аватара — старе видаляється, нове відображається
- [ ] Видалити фото аватара — повертається заглушка

---

## Медичні дані

- [ ] Відкрити вкладку "Медичні дані" — завантажуються збережені дані
- [ ] Змінити групу крові / резус фактор — зберігається
- [ ] Змінити дату першого приступу — зберігається
- [ ] Додати діагноз — зберігається

---

## Ліки

- [ ] Додати новий препарат — з'являється в списку
- [ ] Редагувати препарат — зміни зберігаються
- [ ] Видалити препарат — зникає зі списку
- [ ] Одиниці дози відображаються коректно (таблетки / мл / краплі тощо)

---

## Опікуни

- [ ] Додати опікуна — з'являється в списку
- [ ] Редагувати опікуна — зміни зберігаються
- [ ] Видалити опікуна — зникає зі списку
- [ ] Радіо-кнопки відносин (батько / мати / опікун) — коректно відображаються на обох мовах

---

## Трекінг

- [ ] Відкрити екран трекінгу — завантажуються дані за сьогодні
- [ ] Відмітити прийом ліків — зберігається
- [ ] Заповнити настрій / сон / самопочуття — зберігається після закриття
- [ ] Відкрити трекінг наступного дня — чисті дані (не підтягує вчорашні)

---

## Приступи

- [ ] Додати новий приступ — з'являється в списку та в історії
- [ ] Заповнити всі поля (тип, тривалість, тригери, тяжкість, нотатки) — зберігається
- [ ] Додати відео до приступу — завантажується
- [ ] Редагувати існуючий приступ — зміни зберігаються
- [ ] Видалити приступ — зникає зі списку, відео видаляється з Cloudinary
- [ ] Видалити приступ без відео — видаляється без помилок

---

## Історія

- [ ] Відкрити екран історії — список приступів завантажується
- [ ] Фільтр за тяжкістю — відображає тільки відповідні
- [ ] Календар — приступи відмічені на правильних датах
- [ ] Діаграма по часу доби — відображається коректно
- [ ] Топ тригерів — відображається коректно

---

## Звіт / Експорт

- [ ] Вибрати період — кнопки "від" і "до" працюють
- [ ] Вибрати період без приступів — показується повідомлення про відсутність даних
- [ ] Експорт в PDF — файл відкривається і виглядає коректно
- [ ] Відправка звіту на email — лист приходить з PDF вкладенням
- [ ] Відправка другого звіту одразу — показується повідомлення з датою наступного доступного
- [ ] Звіт на українській мові — всі підписи українською
- [ ] Звіт на англійській мові — всі підписи англійською

---

## Налаштування

- [ ] Перемкнути мову (🇺🇦 / 🇬🇧) — весь текст в додатку змінюється
- [ ] Перемкнути тему (світла / темна / системна) — тема змінюється
- [ ] Зміна каналу зв'язку — зберігається
- [ ] Зміна паролю — успішно, можна увійти з новим
- [ ] Видалення акаунту — акаунт видаляється, перекидає на логін

---

## Перемикач мови на екрані авторизації

- [ ] Прапор 🇺🇦 на екрані логіну — додаток перемикається на українську
- [ ] Прапор 🇬🇧 на екрані логіну — додаток перемикається на англійську
- [ ] Те саме на екрані реєстрації

---

## Загальне

- [ ] Темна тема — ніде нема білого тексту на білому фоні або чорного на чорному
- [ ] Всі екрани в горизонтальній орієнтації — нічого не обрізається (або заблоковано в портреті)
- [ ] Відкрити додаток без інтернету — не падає, показує адекватний стан
- [ ] Довге ім'я / текст у полях — не виходить за межі UI
- [ ] Scroll на всіх екранах з формами — доступний при відкритій клавіатурі
